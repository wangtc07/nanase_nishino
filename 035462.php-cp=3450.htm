わたしは
 
カレー、ステーキ、ソフトクリーム、柿、お好みたい焼き
 
かな
 
 
 
 
完全にいまの気分なだけ、来月同じこと考えたら全選手交代してる気がする、たぶん納豆とか入ってくる

今度出るマガジンの表紙撮影のとき！
 
 
おめめがいたそうなポーズ
 
 
 
 
 
 
いくちゃんいこまと撮ってもらったよん、なかなかない組み合わせ、、貴重、おたのしみに！
 
 
 
 
 
 
 
めっちゃ寒い日だったなー( ･⊝･ )
 
オフショル、まなつみたいね( ･⊝･ )笑
 
 
 
 
 
 
めっちゃ乾燥するね、身体保湿しまくってるよ
手はユースキン、爪はukaのネイルオイル、身体はキュレル、脚はクラランス...現時点で最高のラインナップ...
 
 
 
はあ、なんか、みんなは、秋の夜長を、どうやって、過ごしているの、( ･⊝･ )
 
わたしは、ひたすら、ものを、捨てている( ･⊝･ )
