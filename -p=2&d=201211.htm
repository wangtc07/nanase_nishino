2012/1115Thu西野七瀬海の理由

 
ん～～～～～！ななせまる！！
 
勇者、褒めていただいて嬉しいです。またやります！
 
ちなみにあのコラーゲン鍋めちゃくちゃ辛かったです。初めはいけるんですけど、ていうか食べ続けていたら大丈夫なんですが一度止まると、舌が痺れてきて唾液の分泌は止まらず、涙出そうでした
 
美味しかったです！！
 
 
13日はブログ1周年というわけであらためて自己紹介をしようと思いたった！
 
その14  髪を束ねる
 
名前：西野七瀬
呼び方：ななせまる、なーちゃん、ちそっち
誕生日：1994年5月25日(18歳)
出身：大阪府
血液型 ：O型
 
趣味：絵を描く、写真を撮る、音楽を聴く、ゲームする、漫画読む、ごみの分別
特技：絵を描く、方向音痴じゃないので場所や道をよく覚える
ハマっていること：どうぶつの森、ジョジョ、500円玉貯金、ちゃんちゃんこを着る、お菓子の箱を集める、白ご飯のおかずを考える
 
長所：好きなことへの集中力や執着心が強い！
短所：人見知り、ガラスのハート、すごく行動力がない
 
好きなもの：実家、布団、誕生日、旅行
好きな食べ物：お粥さん、きつねうどん、ねこまんま、豆腐、しめじ
好きなTV番組：旅、クイズ、アニメ
好きな言葉："彼女の瞳が気づかせてくれた  ぼくも愛しているのだと"
好きな動物：犬、猫、鳩
好きな匂い：歯医者、マジックペン、檜、石けん
 
 
こんなものでしょうか～
なんか自分で書き出すと案外思いつかん！
 
1年あっという間
去年の今頃はまだ上京してなくて、たしか13日は、大阪からレッスンを受けに来ててちなみに乃木どこのファッションチェックの収録をしていた日ですな。
「見学だけでも観に行こうか」ってまあやと行って
そしたらなんか自分1位とってるよどうしよう嬉しいとか思ってたら西野呼ばれて、すっぴんの状態でテレビにうつりました。すっぴんデビュー早かった...
あああああ懐かしい！ときがたつのははやい
 
今日まで走ってこれたこと。一人ではたぶん、無理でした
ありがとうございました
これからも良ければ一緒に走ってください。
わたしはよく、深く沈んでしまいますが手を引っ張ってください
 
 
まあやさん。
 
 
 
アリアリアリアリアリアリアリアリ...
アリーヴェデルチ(さよならだ)
 
 
 
このブログにはポニーテールの西野しかいませんが、髪の毛おろしてる西野もぜひ頭に置いておいてね！

2012/1112Mon西野七瀬青い学校

 
ん～～～～～！ななせまる！！
 
勇者ちそっち
どうしてもやってみたかった！ヨシヒコになりたかった！
ヨシーヒコー、ヨシーヒコー
 
 
昨日はハッピーミュージックライブ。初めての横浜アリーナでした
リハの時点で、うわぁぁぁぁ広い！って感じで、さらにランウェイを使ったりといつもと違うアレンジがあったんで、めっちゃ緊張しました！
でもたくさんの方にお越しいただいていと嬉しい気持ち
 
新曲もライブ初披露させていただきました！楽しかった～！
足りない、足りないですね。ステージが前だけやったから後ろの方にも手を振りたかったゼア
しかし遠い場所にいる方々に意識を飛ばしていれば、自然とパフォーマンスは大きくな～るって誰かが言ってた
でも必死になりすぎるとわたしの体がついてこなくて調整が難しい。体力がない上に代謝がいいので汗事情がやばい...
汗をかいてても綺麗ってやっぱりステージ慣れといふものでありぬるかな
 
と、ライブが終わってから、いろいろ考えさせられた。
ありがとうございました！
 
 
その12  DIOの貴様、みているな⁉
一人じゃあれなんで若月さゆにゃんあすかを誘いました。とりあえず反った！
最近ジョジョの画像ばっかり見ている故に...頭の中が承太郎...
 
そういえばまた一泊二日の撮影に行ってたのですが、ほとんどが一人部屋で、のじょあみと隣の部屋ということで、怖いねというわけで、ななの部屋で、さも二人部屋というように過ごしました！
 
その13  ギターを持った写真
ギターのPVのときです。故にベースですが！
 
 
 
質問に答える
 
●洗剤の匂いが好きみたいだけどななが使ってるのはどういうやつ⁇
アタックNEO(緑)
ハミングNEO ベビーパウダーの香り(ピンク)
ワイドハイターEX(緑)
たまにアクロン(ピンク)
うろ覚えやから間違ってるかも！ずっとこのセット使い続けています。
 
●なぁはんは空を飛んだ夢を見たことありますか？
結構あります！あの感覚まだ覚えてます。
 
●ななせまるが好きなおにぎりの具を教えてください！
最近好きなのはいくら！いくらは高級品だと思ふ！
 
●最近幸せを感じた出来事は？
普段、なーちゃんと呼ぶまいやんが、たまに七瀬と呼んでくれる( ；∀；)ぴぎゃー
 
●やっぱり生写真は集めてほしいものですか？
うん！でも飽きられるのは怖いよ...(　ﾟдﾟ　)

2012/1109Fri西野七瀬ひざのうえのたむらまろ

ん～～～～～！ななせまる！！
 
 
ガールズアワード
昨日、はじめて出演させていただきました(　ﾟдﾟ　)
乃木坂46のパフォーマンスをしたあと、まいやんと西野がモデルとして、ランウェイを歩かせていただきました！！
 
ライブは女の子ばっかりで照れました！
あと指差して、とか投げkissして、っていううちわがいっぱいでなんか新鮮！！しかしめっちゃ楽しかったです(　ﾟдﾟ　)
 
終わったらヘアメイクをしてもらって着替えて...
洋服はISBITさんです。めっちゃ可愛いいいいい(　ﾟдﾟ　)
しかも紫で乃木カラー！
此れは歩き終わった直後！にんじん笑顔！
リボンが外れてる...しかし歩いてるときはちゃんと結んでました！
 
生で見るモデルさん達のスタイルの良さは生きる次元が違います(　ﾟдﾟ　)(　ﾟдﾟ　)
 
歩く練習ちょっとだけ指導してもらったんですけどめっちゃ難しくて、足元ばっかり見てしまって不安いっぱいでした
でも乃木坂46として歩かせてもらうから、私たちらしく歩こう！とまいやんと決めました
(*＞0＜*)
 
出番前、まいやんと上下そでに別れてから一人で待つ時間が長く、頭の中では段取りをずっとシミュレーション...。隣には綺麗なモデルさん達が写真を撮られている、と思ったら一緒に撮ろうと呼んでくださり嬉しい気持ち！
「これ客席にポーンと投げて下さいね」と言って渡された可愛いマスコット？をにぎにぎ握ってる間も、かっこいい音楽に合わせて重低音がお腹に響く(　ﾟдﾟ　)
 
結果、やっぱり緊張しすぎて体が固かった！でもいつもと違う、たくさんの女の子のお客さんが手を振ってくれて、めっちゃ嬉しくて楽しくて気持ちよく歩き終わることができました！！
終わってからメンバーが「おかえり、よかったよ～！」って駆け寄ってくれてなんか涙が出そうでした
 
まいやんと出演することができてめっちゃ良い経験になったし、また機会をいただける事ができたら是非ともリベンジしたいなと思ひまつる！
ガールズアワードさんありがとうございました！！
 
まいやんもななもツインテールなふ！
 
 
とびだせどうぶつの森がお家に届いてるぜェーーっ！まま予約ありがとう！
やろうかなあ...ワクワク...と思いつつ、にっぽんのミンイを見ています。この番組も結構好き！リアルタイムで投票するやつ
投票はしたことないけどYESかNOかどっちかなぁって考えながらぼーっと見るのがよきかな(　・´ｰ・｀)
 
あ、お湯はりを押さずに蛇口回しっぱなしにしていた...お湯溢れてる
やれやれだぜ
ちょっとブレた
 
 
 
 
質問に答える
 
●生駒ちゃんはななせを単推ししてるみたいだけど、日ごろからアプローチしてきたりするんかな？
最近ななのほっぺたを触ってくれます。生駒ちゃんルール的には「なーちゃんにはあんまりベタベタ触りすぎない」とのことです
良い距離を保って、大事にされています！！
 
●なぁはんは白いヘビやグリーンヘビは好きですか？
ヘビ好き！故にペットショップ行くといつも観察してしまう
 
●来月はエヴァンゲリヲンの最新作が公開されるんですが、なぁちゃんはエヴァンゲリヲンは見たことがありますか？
昔から興味があったんです！
だから今、再放送とかしてるからちゃんと見ようと思って録画しましたえへ
 
●コンタクト入れるコツがあるならぜひ教えて下さい！！
コツっていうか慣れやと思います
中1からコンタクトにしたけど初めは全然入れられへんし取ることもできひんしって生活でしたので...
ちなみに洗浄液はRenuがめっちゃオススメ！開封したてみたいな潤いがすごい！
 
●ヤクルトは好きですか？
大好きです！！
でも甘い故に飲みすぎはいと注意しにけり
 
 
おはよう！
昨日の夜に書いてたブログを送信し忘れて寝てしまったパターンです
いつも読んでいただいてる方、コメントくださる方、ありがとうございます。
西野七瀬がんばります
勉強やお仕事、みなさん今日も一日がんばりましょうヽ(｡>▽<｡)ﾉガンバレ
