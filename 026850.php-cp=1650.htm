2015/1204Fri橋本奈々未クリスマスだね

 
みんな見てくれた？今週発売のスピリッツ！
表紙と巻頭グラビアさせてもらってるんだけど、とてもお気に入り！みてない人は急いで見に行ってみて٩(๑`^´๑)۶
 
 
 
アザーカット！
 
スピリッツさん、ありがとうございました！！
 
 
 
 
 
 
 
最近色んな撮影をしてるよ！
 
今月２４日発売になるストリートジャックも表紙をさせてもらいます！
昨日撮影行ってきたよ！楽しい撮影だったー٩(๑`^´๑)۶
こちらもなかなか他ではやらない設定でのものだったので、お楽しみに(﹡´◡`﹡ )
 
 
けん玉あったからやった！
 
もしかめしたらみんな おぉうまい！って言ってくれる！笑
 
でもわたしの小学校の同級生ほとんどみんなもしかめ出来てた気がする...
みんな練習すればできるので、わたしが特別けん玉が得意で上手いわけではないのだ...
 
しかしそんなことは、ほめてくれるスタッフさんの前ではえへへって笑って黙っておいた 
すみません 笑
 
 
 
 
 
 
 
 
いくたどん、大阪にいる、がんばれいか
 
 
 
 
髪伸びてきた どうしよーどうしたいっていう欲がないからとりあえず放置してるんだよーどうしよー(　ﾟДﾟ　)
 
 
 
 
 
 
 
あ！明日珍プレー好プレー大賞！！21時からフジテレビ！
わたしも審査員として参加させてもらっているので是非みてね！面白かった！！！
 
 
 
 
じゃ、体のメンテナンスに行ってくるぜ！！
