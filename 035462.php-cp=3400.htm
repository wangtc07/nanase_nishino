最近食べたいものはないんだけど食欲が止まらないの、そんなことある？
とりあえず口に入れたくなる、、、
 
いまはオムライス食べてるんだけど、みんなはこの先の人生で５品だけをヘビロテして生きていかなきゃいけないとしたら、なにを選ぶ？
 
 
 
 
わたしは
 
カレー、ステーキ、ソフトクリーム、柿、お好みたい焼き
 
かな
 
 
 
 
完全にいまの気分なだけ、来月同じこと考えたら全選手交代してる気がする、たぶん納豆とか入ってくる
 
 
 
 
 
にしても今のわたしのオールスター、だいぶ胃が若いよね、、、こんなこと書いてるうちにオムライス食べ終わってチョコブラウニーパフェ食べてるよ、ああどこのお店にいるか分かる人は分かりそう
 
久々にパフェというものを食べたわ！
 
 
 
 
 
 
スタジオに何ていう名前か忘れたけど懐かしいやつがいた！
 
 
今度出るマガジンの表紙撮影のとき！
 
 
おめめがいたそうなポーズ
 
 
 
 
 
 
いくちゃんいこまと撮ってもらったよん、なかなかない組み合わせ、、貴重、おたのしみに！
 
 
 
 
 
 
 
めっちゃ寒い日だったなー( ･⊝･ )
 
オフショル、まなつみたいね( ･⊝･ )笑
 
 
 
 
 
 
めっちゃ乾燥するね、身体保湿しまくってるよ
手はユースキン、爪はukaのネイルオイル、身体はキュレル、脚はクラランス...現時点で最高のラインナップ...
 
 
 
はあ、なんか、みんなは、秋の夜長を、どうやって、過ごしているの、( ･⊝･ )
 
わたしは、ひたすら、ものを、捨てている( ･⊝･ )
 
年末大掃除いらずレベル( ･⊝･ )ふふ、、、
