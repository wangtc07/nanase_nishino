2012/1022Mon西野七瀬ぱわ

 
ん～～～～～！ななせまる！！EX
 
近いっしょ！
htcの使い方がわからずあーやこーややってたらシャッター押してまうし近すぎてピンボケしたの図です
 
 
また考え事をしてしまう時期到来(　ﾟдﾟ　)
どうしよう。定期的にくる
コンビニでいつもは絶対に買わないような高めのお菓子とか買いたくなる
あ、すでに昨日買ってしまった...冷蔵庫にはいってる
しかもチョコのアイスとチョコのお菓子！なんで？二個もチョコって！な  ん  で  や  。
 
 
あ！きいてください！
電車にのってたら
荷物をたくさん持った、外国人さんが座ってて、かばんの整理してはって硬貨が落ちたんです。日本円じゃないからすぐその人のんって分かったけど外人さん、気づかず(　ﾟдﾟ　)
勇気を出して、駅にとまったときに拾って渡しました。立ち上がるまでめっちゃ緊張しました(　ﾟдﾟ　)ドキドキ
でも笑顔で受け取ってくれたし良かったーー！
 
 
その1
カメラをみる
 
 
 
昨日は乃木どこで、ロケに行きました。いいですね外は。天気も良かったです！
帰宅してすばやく寝る準備をしてたら、11時くらいに寝てしまい乃木どこを見れませんでした。帰ったら見ようぞ！！
 
おもしろかったかな？
 
 
質問に答える
●どいやさんは100m何秒くらいで走りますか？笑
7.7秒
 
●なぁちゃんの一番のお気に入りのご当地ご飯ってなにー？
まだ多くは食べてないので知らないんですが、九州で食べた水炊きはめっちゃ美味しかったです。
 
●鳥系のポケモンでなにが一番好き？ちなみに俺はフリーザーとムクホーク
ん～～～～～！チルタリス！！
 
●ストール巻く男子どう思う?
いと良しと思ふ(*＞0＜*)でも個人的にはマフラーの方が好きかなえへ
 
●100万円 貯金無しでやったら何に使う？
80万は両親にあげます。残りは...
んー。足のマッサージに行ったりタンスを買ったりしたい！あと冬のコートだ！あとカメラ買いたい！
 
 
 
ジョジョ立ちを極めようとおもひます！
 
幸せをありがとう
