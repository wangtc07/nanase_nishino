2012/1118Sun西野七瀬水兵ねずみ

 
ん～～～～～！ななせまる！！
 
 
まいまい、かずみん、みゅうみゅう、あみとマカオにいってきました！
ただいまー！！
 
乃木坂として西野の初の海外、マカオグランプリのお仕事です。
はじめは難しいなーと思ってたんですが、実際に行ってみていろいろと学習してきたのて自分なりに、まとめます！！
 
車のレースで、街中がサーキットになっていて、今回のマカオグランプリは特に難しいといわれている！
2日間にわたってマカオには世界中の強者が集まりました
私たちが応援させてもらえることになったので、もっとたくさんの方にレースに興味を持っていただけたらいいなぁという思ひです！
 
さらにケイ・コッツォリーノ選手のマシン(46番)にわたしたちの顔をデザインして、レースに参加していただきました(*＞0＜*)
純粋に、嬉しかった！あと不思議！
 
席に座ってみると、たくさんのマシンがピットに戻ってきてタイヤ交換していたり、マシンが目の前をすごいスピードと音で駆け抜けていきます。迫力ッ
これ、結構な音量なんですがずっと観ているとわたしは居心地が良かったです(　ﾟдﾟ　)
楽しく応援しました～！
 
あとあとマカオは初めてだったんですがめっちゃキラキラした街でした！
そしてまたもや一人部屋だったんで、怖いからあみと二泊三日過ごしました(笑)
 
 
 
 
あ、コメントでTwitterにアカウントがあるとたまに見かけるんですが...
乃木坂46は登録してません！
(みさみさだけはミスマガとしてしてます)
故にニセモノにご用心(　ﾟдﾟ　)
 
 
 
質問に答える
 
●どっさんに関係なくても、なんか絵のリクエストしても...いいかなぁ？φ(.. )
もちろんです。遅いですがばっちこい！
 
●サンタさん信じてる？
いつからか見破りました！がしかしサンタさんの存在は今もインマイハート
 
●僕は、子どもん時から海老がどうしても食べられへんねんけど、どないしたらええでっしゃろ？
食べれないものは食べれないままでいいと思います。しかし私は海老は好きです！美味しい！
 
●なぁちゃんは暖房とこたつとストーブこの３つの中で暖まるのに一番使うやつはどれですか？
実家のときはこたつ、今は暖房です。
 
●どっさんって服着てないけど寒くないのかなあ?
寒がってます！
故にニットとか着てます！
 
 
その15  ウインク
自分で撮るのはいと難しかった...
今日は似顔絵会と、カレンダーお渡し会！頑張ろ！
 
 
 
アリーデウェルチ(　ﾟдﾟ　)
