2012/1120Tue西野七瀬ボラボラボラ

 
ん～～～～～！ななせまる！！
 
雲の上で会おうか
 
 
似顔絵会、カレンダーお渡し会にお越しいただいたみなさん
ありがとうございました(　ﾟдﾟ　)
似顔絵は去年も参加させていただいたんですが制限時間5分ということで結構、恥ずかしかったです。大事にしていただけたらいと嬉し！いと幸い！
 
カレンダーはねねころとペアでした！
お渡し会って面白いなーと新たな発見をしました。時間ないからあんまり喋られへんけど。
またやりたい
結果、あんなけ可愛い子がたくさん載ってるカレンダーなかなか無いな、と思った(　ﾟдﾟ　)
 
似顔絵会はゆみ姉にとって最後のイベントでした
終わってからホワイトボードにみんなでいろいろ書いて、ゆみ姉を呼んで、「お疲れ様」って言った。花束を渡して、みんなのメッセージをまとめた本を渡して、ケーキを食べました！
なんか...ほんまに最後って感じがしませんでした
一緒にゲームで通信もしてたし...
いつもと同じ笑顔でした
ゆみ姉の笑顔好き！いい笑顔だなぁっていつも思ってた
 
いつまでもみんなのゆみ姉！
大好きです！！
 
 
 
そういえばこないだ健康診断に行ったんですが、採血が初めてで、手首を触られると生命の危機を感じるわたしにとっては恐怖でした。
しかも実物の血液はダメだ(　ﾟдﾟ　)
怖いです怖いですって言ってたら優しいおばちゃんが「赤ちゃん用の針でしよっか、ねえ」って言ってくれてベッドに寝ながら赤ちゃん用の細い針で採血してもらいました...
おかげさまで痛くありませんでした...
ありがとうございました
 
 
 
質問に答える
 
●ラジオのパーソナリティになったらどんなこと話したい？
くだらない話をだらだらしたい
はたしてどこまでついてこれる人がいるのか。
 
●どっさんは、右利き左利き?
王道のサウスポー
 
●もしタイムマシンが1回だけ使えたら未来、過去どの時代に行きたい？行って何する(●･ч･)？
未来の様子！
でも見たいけど考えてみてもワクワクしないしむしろ悲しいなあ......
 
●七瀬がクリスマスに聴きたい曲は？
いつもCMで流れてる曲とか、お店で流れてる曲とか、聴くだけで満足！
 
●なあちゃんと同じくらいどっさんのことも好きなんだけどどうしたらいい？(笑）
えー。嫉妬(　ﾟдﾟ　)やれやれだぜ
 
 
 
その16  手でつくったキツネときす
 
 
ジョジョの5部10巻セットみたいなの買ってしまったー(　ﾟдﾟ　)どっひゃー
はやく届けー(　ﾟдﾟ　)
 
アリーヴェデルチ！
